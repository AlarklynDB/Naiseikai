# 内省界 | The Naiseikai Universe Website

Official universe hub for The Naiseikai Universe by Alarkius Elvya Jay.

## Tech Stack
- React + TypeScript + Vite
- Tailwind CSS (custom night skyline theme)
- React Router DOM
- Cloudflare Workers + Pages deployment

## Pages
- `/` — Home (hero, intro, quick nav)
- `/universe` — Universe Overview (themes, genres, Ma philosophy)
- `/lore` — Lore & Timeline (world history)
- `/worldbuilding` — Worldbuilding (races, setting, factions)
- `/titles` — Titles & Stories (all LN titles in story order)
- `/songbook` — The Naiseikai Songbook
- `/author` — Author's Note & Content Policy

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy (Cloudflare Workers)
```bash
npm run build
npx wrangler deploy
```
